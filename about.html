About The Site

This site was made for my finals on introduction to computing. I used VSCode IDE to edit HTML and CSS, as well as Open-props to style the elements.

Styling

Open-props is a very modular CSS framework. It basically just provides the values that you can then use to style your elements.

Color scheme

The color scheme I used is called Biscuit. It's a warm and cozy dark colorscheme, which is perfect for what this site needs.

Typography

For the font, I used HK Grotesk. It's a friendly but modern font, and kind of blends well with the color scheme. I also used Transfonter to easily generate the font-kit to use for the website.
